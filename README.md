## Islamic Counter

This is a simple Islamic counter app built with React and Vite.

## Features

- Count the number of times you perform a tasbih
- Add custom dhikr
- Add custom transliteration
- Add custom meaning
- Add custom color theme
- Add custom vibration and sound feedback


## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```
